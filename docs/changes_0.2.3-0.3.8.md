# 0.2.3 → 0.3.9 更新概要

## 可视化功能调整
`Visualizer` 标签页用于查看运行输出。打开后会自动定位到最近一次运行的输出目录，并加载 `*.txt` 文件列表。对 `Phimax_*.txt` 等快照数据启用 `Plot` 热图（滚轮缩放、鼠标拖动、Fit、PNG 导出）；其余文件默认使用 `Table` 查看。

## 文本输出解析与展示
对常见输出补充了专用显示：`run_config.txt` 以 `key/value` 两列展示，便于核对参数；`checkpoint_timestamps.txt` 在保留 `timestamp_ms` 的同时补充可读时间、累计耗时（elapsed）与相邻间隔（delta），便于评估耗时与波动。

## Sweep 扫参模式重构
批量扫参从固定参数集改为“逐参数配置”。每个参数可选 Fixed / Range / List：固定值、起止步长生成序列、或逗号输入指定列表；同时提供预览与组合数量提示，减少配置错误。

## 运行与交互改进
单次与批量运行增加 step 级进度条与总进度显示。界面布局与样式进行了多次调整；同时禁用单次/扫参页面参数输入框的滚轮改值，避免误修改。

## 打包与细节完善
应用与发布产物命名统一为 `MID Nano`。并对跨平台细节问题（目录选择/导出对话框兼容性、表格列宽、标签页边线等）进行修复与优化，关键交互处补充日志提示，便于定位问题。
