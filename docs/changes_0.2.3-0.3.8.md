# 0.2.3 → 0.3.8 更新概要

1. **可视化（Visualizer）更好用**：打开标签页会默认定位到“当前输出目录”，左侧自动加载 `*.txt` 列表；只有 `Phimax_*.txt` 这类快照会启用 `Plot`（热图渲染、`Ctrl+滚轮` 缩放、Fit、导出 PNG），其他文本默认走 `Table` 查看原始数据。
2. **输出文本显示更友好**：对 `run_config.txt` 自动按 `key/value` 两列展示；对 `checkpoint_timestamps.txt` 自动补充时间、耗时（elapsed）和间隔（delta）等字段，方便直接看性能与进度。
3. **Sweep 扫参能力增强**：从“写死参数”升级为“每个参数都可选 Fixed / Range / List”，Range 可用开始/结束/步长生成，List 支持逗号输入指定列表，并提供预览与组合数量提示，更容易上手也更不容易配错。
4. **运行与交互体验提升**：单次/批量增加 step 进度条与总进度；界面间距与配色做了多轮微调；同时禁用第一、第二标签页参数框的滚轮改值，避免滚动页面时误触修改参数。
5. **打包与细节修复**：应用与发布产物命名统一为 `MID Nano`；修复并加强了 `Browse…` / `Export PNG…` 等按钮在部分环境下不生效的问题（必要时会在日志里输出提示），并去掉可视化页顶部多余的分割线、优化表格列宽显示。
