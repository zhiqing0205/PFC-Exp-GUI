# GUI 标签页功能说明

## 1. Experiment（实验/扫参）

该标签页用于配置并运行实验，同时也覆盖“单次实验”和“批量扫参”的场景。每个参数都可以选择 `Fixed / Range / List` 三种方式生成取值序列：`Fixed` 固定一个值、`Range` 通过 `Start / End / Step` 生成并在右侧预览为列表、`List` 允许直接输入逗号分隔的数值列表。系统会按笛卡尔积组合生成任务队列并顺序运行：当所有参数都为 `Fixed` 时，总运行数为 1（等价于单次实验）；当任意参数使用 `Range/List` 且产生多值时，会自动运行多组组合实验。运行时会显示“当前 run 的 step 进度”和“整体 run 进度”，同时把每组运行的命令与输出目录写入底部状态栏（Log）。单击状态栏可打开独立的 Log 窗口查看完整输出。你可以随时 `Stop` 中断队列并保留已完成结果，或点击 `Open Output` 直接打开当前输出目录。

## 2. Visualizer（输出可视化）

该标签页用于查看输出目录中的 `*.txt`。打开标签页时会默认定位到最近一次运行的输出目录并加载文件列表，也可以通过 `Browse…/Refresh` 手动切换与刷新。

- 对一般文本输出：默认使用 `Table` 表格预览（可复制/筛选，便于粘贴到 Excel/Origin 等进一步分析）。
- 对 `run_config.txt`：以 `key/value` 两列展示参数，便于核对本次运行配置。
- 对 `checkpoint_timestamps.txt`：展示 `timestamp_ms`、可读时间、累计耗时（elapsed）与间隔（delta），便于快速看性能与进度。
- 对 `Phimax_*.txt / Phimaxq4q6_*.txt`：启用 `Plot` 热图渲染（支持滚轮缩放、鼠标左键拖动、Fit 自适应与导出 PNG）。

## 3. Manufacturing / Transformation / Stress–Strain（规划中）
