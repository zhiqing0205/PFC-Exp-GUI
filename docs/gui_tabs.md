# GUI 标签页功能说明

## 1. Single Run（单次实验）

该标签页用于快速配置并运行一次实验。你可以在「Parameters」中调整关键参数（如密度/序参量均值 `u0`、浓度 `con0`、核参数 `sig`、时间/空间步长 `dt/dx`、迭代步数 `steps`、输出间隔 `mod`、随机种子 `seed` 等），在「Output」中选择输出根目录并可选自定义本次运行的文件夹名（留空则自动使用时间戳命名）。点击 `Run` 后，GUI 会调用同目录下的 `pfc-exp-cli` 执行并把结果写入输出目录，同时写入 `params.json / run_config.txt` 便于复现实验。运行过程与关键日志会实时显示在下方 Log 区域；需要中止时可用 `Stop` 终止当前任务。你也可以拖动分隔条调整 Log 高度，或点击 `Open Output` 直接打开结果目录。

## 2. Batch Sweep（批量扫参）

该标签页用于做多组组合实验，适合用来探索参数敏感性。Batch Sweep 采用“逐参数模式”：每个参数都可以选择 `Fixed / Range / List` 三种方式生成取值序列，其中 `Range` 通过 `Start / End / Step` 生成并在右侧预览为列表，`List` 允许直接输入逗号分隔的数值列表。系统按笛卡尔积组合生成任务队列，并在界面中给出预览与总运行数；运行时显示“当前 run 的 step 进度”和“整体 run 进度”，同时把每组运行的命令与输出目录写入 Log。你可以自定义 batch 名称，输出目录会自动按组合依次创建。每个组合会在 batch 目录下创建独立子文件夹保存输出与参数快照，方便后续筛选、对比与复现实验；你也可以随时 `Stop` 中断队列并保留已完成结果。

## 3. Visualizer（输出可视化）

该标签页用于查看输出目录中的 `*.txt`。打开标签页时会默认定位到最近一次运行的输出目录并加载文件列表，也可以通过 `Browse…/Refresh` 手动切换与刷新。

- 对一般文本输出：默认使用 `Table` 表格预览（可复制/筛选，便于粘贴到 Excel/Origin 等进一步分析）。
- 对 `run_config.txt`：以 `key/value` 两列展示参数，便于核对本次运行配置。
- 对 `checkpoint_timestamps.txt`：展示 `timestamp_ms`、可读时间、累计耗时（elapsed）与间隔（delta），便于快速看性能与进度。
- 对 `Phimax_*.txt / Phimaxq4q6_*.txt`：启用 `Plot` 热图渲染（支持滚轮缩放、鼠标左键拖动、Fit 自适应与导出 PNG）。

## 4. Manufacturing / Transformation / Stress–Strain（规划中）

后三个标签页为后续功能预留入口：制造/涂层工艺、结构转变分析、应力-应变加载与曲线输出等。目前仅提供路线图与说明文案，后续会逐步补齐对应参数面板、数据处理与可视化流程，并尽量保持“能直接上手”的使用体验，也欢迎按需求扩展。
